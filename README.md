# Lacrei Saúde - Desafio Cypress

Este repositório contém a solução para um desafio técnico proposto pela **Lacrei Saúde**, uma plataforma que conecta profissionais da saúde à comunidade LGBTQIAPN+.

O objetivo do desafio foi realizar a **automação do fluxo de cadastro de usuário** utilizando o framework de testes **Cypress**.

Os testes foram executados no ambiente de staging da aplicação:

https://paciente-staging.lacreisaude.com.br/

---

## Cenários Automatizados

Foram automatizados cenários relacionados ao fluxo de cadastro, incluindo:

* Cadastro com dados válidos
* Validação de campos obrigatórios
* Verificação do comportamento do sistema após a finalização do cadastro
